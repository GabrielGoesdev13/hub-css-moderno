Desvendando o CSS Grid

O CSS Grid Layout é um sistema de layout bidimensional (2D). Ele permite dividir a página em regiões principais ou definir a relação (em tamanho, posição e camada) entre partes de um controle HTML.

Definindo a Grade (O Contêiner "Pai")

Para começar, aplicamos display: grid; ao contêiner "pai".

  • grid-template-columns: Define o número e o tamanho das colunas. (Ex: repeat(3, 1fr) cria 3 colunas de tamanho igual). A unidade fr (fração) distribui o espaço disponível.
  • grid-template-rows: Define o tamanho (altura) das linhas. (Ex: auto ou 100px).
  • gap (ou grid-gap): Define o espaçamento (canaleta) entre as células do grid. (Ex: 15px).

Posicionando Itens na Grade (Os "Filhos")

O poder do Grid está em posicionar os "filhos" exatamente onde queremos, fazendo-os ocupar uma ou mais células.

  • grid-column-start / grid-column-end: Define em qual linha de coluna o item começa e termina.
  • grid-row-start / grid-row-end: Define em qual linha de linha o item começa e termina.
  • Shorthand (Atalho):
    • grid-column: 1 / 3; (Ocupa da linha 1 até a 3)
    • grid-column: 1 / span 2; (Começa na linha 1 e ocupa 2 colunas)
