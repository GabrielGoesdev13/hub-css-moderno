Bem-vindo(a)!

Este é o Hub de estudos de CSS Moderno. Nosso objetivo é explorar as ferramentas mais poderosas para a criação de layouts responsivos e complexos de forma eficiente.

Use a navegação acima para explorar os conceitos de Flexbox (para layouts de uma dimensão) e Grid (para layouts de duas dimensões).

O que você vai aprender:

  • Flexbox: Como alinhar, distribuir e ordenar elementos em um contêiner.
  • Grid: Como criar sistemas de "grade" complexos com controle total sobre linhas e colunas.
