Desvendando o Flexbox

O Flexbox (Flexible Box Layout Module) é um modelo de layout unidimensional projetado para distribuir espaço e alinhar itens dentro de um contêiner, mesmo quando o tamanho desses itens é desconhecido ou dinâmico.

Conceitos Básicos (O Contêiner "Pai")

Tudo começa aplicando display: flex; ao elemento "pai" (o contêiner).

  • flex-direction: Define o eixo principal. row (horizontal, padrão) ou column (vertical).
  • justify-content: Alinha os itens ao longo do eixo principal (ex: center, space-between, flex-start).
  • align-items: Alinha os itens ao longo do eixo transversal (o eixo oposto, ex: center, stretch).
  • flex-wrap: Permite que os itens "quebrem" para a próxima linha se não couberem. Padrão é nowrap.

Conceitos Intermediários (Os Itens "Filhos")

Podemos controlar o comportamento individual de cada item "filho" dentro do contêiner flex.

  • flex-grow: Define a capacidade de um item "crescer" para ocupar o espaço livre. (Ex: flex-grow: 1;)
  • flex-shrink: Define a capacidade de um item "encolher" se não houver espaço suficiente.
  • flex-basis: Define o tamanho inicial de um item antes que o espaço livre seja distribuído.
  • flex: A propriedade "shorthand" (atalho) para flex-grow, flex-shrink e flex-basis. (Ex: flex: 1;)
  • align-self: Permite que um item individual sobrescreva o align-items definido no "pai".
